✨ **Via Veritatis** é um projeto pessoal criado com o objetivo de desenvolver minhas habilidades para virar **fullstack** 💻, ao mesmo tempo em que aprofundo meus estudos sobre a fé católica ✝️.

A proposta é construir um site dedicado à publicação de artigos, documentos e conteúdos de catequese 📚, unindo aprendizado técnico com crescimento espiritual. Mais do que um projeto de programação, o Via Veritatis também é um espaço de formação e busca pela verdade.

🚧 O projeto está em constante evolução e serve como base prática para aplicar conceitos de front-end e back-end, organização de dados e estruturação de sistemas.

🔮 Futuramente, o site contará com novas funcionalidades, como sistema de busca, categorização de conteúdos, melhorias na experiência do usuário e possíveis recursos interativos.

A ideia é que o **Via Veritatis** cresça continuamente, acompanhando meu desenvolvimento como programador 👨‍💻 e também minha caminhada na fé 🙏.
